# スーパーTANABEブラザーズ 〜バルーンファイト〜

OpenGame + gpt-5-mini で生成した、**バルーンファイト**風2人対戦アクションゲーム。

## 概要

2人の田邊兄弟（タケシ＆ケンジ）が風船で空に浮かびながら戦う、1984年名作「バルーンファイト」オマージュ作品。

**兄・タケシ（メガネ）**：黒ぶちメガネ、ボサボサ髪、無精ひげ
**弟・ケンジ（おかっぱ）**：おかっぱ頭、左頬のほくろ、カーディガン姿

## 操作方法

| 操作 | 兄（タケシ） | 弟（ケンジ） |
|------|-------------|-------------|
| 移動 | A/D | ←/→ |
| 浮く（パタパタ） | W | ↑ |
| スマホ | 左半分タップ | 右半分タップ |

## ルール
- 画面上を風船で浮いて移動
- 相手に体当たりして風船を割れ！
- 風船が0になると落下→ミス
- 敵（CPU）の風船も割れる！
- 最後まで生き残れ！
