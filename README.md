# スーパーTANABEブラザーズ (Super TANABE Brothers)

OpenGame + gpt-5-mini で生成した2人同時協力プレイのマリオ風横スクロールアクションゲーム。

## 操作方法

| 操作 | 兄（タケシ） | 弟（ケンジ） |
|------|-------------|-------------|
| 移動 | WASD | 矢印キー |
| ジャンプ | W | ↑ |
| スマホ | 左半分タップ | 右半分タップ |

## 遊び方

- 2人の田邊兄弟を操作してステージを進もう
- 兄・タケシ（メガネ）は走るのが速く二段ジャンプ可能
- 弟・ケンジ（おかっぱ）はジャンプが高く壁キック可能
- 敵を踏んで倒し、コインを集めてハイスコアを目指せ！
